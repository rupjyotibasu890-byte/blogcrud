Blog CRUD Application

Live Demo: https://simpledatabase.xo.je/blogcrud/

Description

A simple blog system built with PHP and MySQL where users can:

Create blog posts using a secret codename.

View all blog posts (newest first) in readable boxes.

Edit or delete a blog post only if they know the correct codename.

Delete operations include a confirmation prompt.

Features

Create, Read, Update, Delete (CRUD) functionality.

Each blog post has a unique secret codename for security.

Blog list styled for readability and user experience.

Uses prepared statements for database security.
